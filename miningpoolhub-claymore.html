Как настроить Claymore для Miningpoolhub

Перейдите на сайт https://miningpoolhub.com чтобы зарегистрировать аккаунт.

Шаг 1. создайте или отредактируйте .bat файл

Пример

EthDcrMiner64.exe -epool europe.ethash-hub.miningpoolhub.com:20535 -ewal rgzrgz_leo.rgzrgz_leo -epsw x -mode 1 -dbg -1 -mport 0 -etha 0 -ftime 55 -retrydelay 1 -tt 79 -ttli 85 -tstop 89 -esm 2
pause

Так выглядит адрес пула в пакетном файле прогрммы

Просто вставьте сюда Ваш коешелк

EthDcrMiner64.exe -epool europe.ethash-hub.miningpoolhub.com:20535 -ewal ВАШ_КОШЕЛЕК -mode 1 -dbg -1 -mport 0 -etha 0 -ftime 55 -retrydelay 1 -tt 79 -ttli 85 -tstop 89 -esm 2
pause

Выбор сервера Miningpoolhub

Для выбора наиболее подходящего для вас сервера в зависимости от вашего местоположения используйте:

Расположение Адрес Стратум порт
Европа europe.ethash-hub.miningpoolhub.com 20535
США us-east.ethash-hub.miningpoolhub.com 20535
Азия asia.ethash-hub.miningpoolhub.com 20535

В этом примере мы использовали европейские сервера, так что теперь внутри вашего текстового файла вы должны иметь:

setx GPU_FORCE_64BIT_PTR 0

setx GPU_MAX_HEAP_SIZE 100

setx GPU_USE_SYNC_OBJECTS 1

setx GPU_MAX_ALLOC_PERCENT 100

setx GPU_SINGLE_ALLOC_PERCENT 100

EthDcrMiner64.exe -epool europe.ethash-hub.miningpoolhub.com:20535 -ewal Ваше_имя_пользователя_пула.Ваше_имя_воркера -epsw x

Замените «Ваше_имя_пользователя_пула» на свое имя пользователя пула miningpoolhub скопируйте его из другого текстового файла и вставьте его здесь.

Так выглядит username и workername во вкладке Hub Workers

Замените "Ваше_имя_воркера" на имя воркера, который вы создали на пуле miningpoolhub:

setx GPU_FORCE_64BIT_PTR 0

setx GPU_MAX_HEAP_SIZE 100

setx GPU_USE_SYNC_OBJECTS 1

setx GPU_MAX_ALLOC_PERCENT 100

setx GPU_SINGLE_ALLOC_PERCENT 100

EthDcrMiner64.exe -epool europe.ethash-hub.miningpoolhub.com:20535 -ewal rgzrgz_leo.rgzrgz_leo -epsw x -mode 1 -dbg -1 -mport 0 -etha 0 -ftime 55 -retrydelay 1 -tt 79 -ttli 85 -tstop 89 -esm 2
pause

Сохраните ваш файл как пакетный файл с расширением bat. Для этого выберите «Файл»> «Сохранить как», а затем в поле выберите его имя и введите «.bat» после имени. В поле «Тип файла» выберите «Все файлы» и нажмите «Сохранить».

Поздравляем! Теперь у вас есть готовый пакетный файл для Miningpoolhub! Создайте ярлык для нового Bat файла и отправьте его на рабочий стол.
