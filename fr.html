Démarrage rapide

Étape 1: Téléchargez le miner

Une fois que vous aurez téléchargé le fichier, décompressez l’archive et trouvez le fichier start.bat. Ensuite, effectuez un clic droit puis Éditez, un fichier texte devrait s’ouvrir et allez paramétrer Claymore.

Attention! Pour des raisons de sécurité, Windows peut vous empêcher d'ouvrir le fichier bat. Dans ce cas, vous devez l'autoriser à s'ouvrir dans une fenêtre contextuelle.。

Étape 2：Configurer le fichier .bat pour exécuter Claymore Dual Miner

setx GPU_FORCE_64BIT_PTR 0
setx GPU_MAX_HEAP_SIZE 100
setx GPU_USE_SYNC_OBJECTS 1
setx GPU_MAX_ALLOC_PERCENT 100
setx GPU_SINGLE_ALLOC_PERCENT 100
EthDcrMiner64.exe -epool eth-eu2.nanopool.org:9999 -ewal VOTRE_WALLET -eworker NOM_DU_WORKER -epsw x -mode 1
setx : pour des paramètres de configuration du GPU
 -epool eth-eu2.nanopool.org:9999 : pour utiliser nanopool pool (vous pouvez en choisir un autre si vous voulez)
-ewal 0x360d6f9efea21c82d341504366fd1c2eeea8fa9d : l’adresse de votre wallet. METTEZ LE VOTRE !
-epsw x : mot de passe du pool si nécessaire
-eworker : un nom arbitraire que vous voulez donner à votre worker

Étape 3:Lancez

Lancez le logiciel en double-cliquant sur votre fichier START.bat，Enregistrez et lancez le fichier. Le minage se lance.
