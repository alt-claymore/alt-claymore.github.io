Как настроить Claymore для Dwarfpool

Шаг 1. создайте или отредактируйте .bat файл

Пример

EthDcrMiner64.exe -epool eth-ru2.dwarfpool.com:8008 -ewal 0x9147460980c93629e775783148591b7d0a0cbf2d -eworker Claymore -mode 1 -r 0 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -ftime 55 -tt 79 -ttli 85 -tstop 89 -fanmin 30

pause

Так выглядит адрес пула в пакетном файле прогрммы

Просто вставьте сюда Ваш коешелк

EthDcrMiner64.exe -epool eth-ru2.dwarfpool.com:8008 -ewal ВАШ_КОШЕЛЕК -eworker Claymore -mode 1 -r 0 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -ftime 55 -tt 79 -ttli 85 -tstop 89 -fanmin 30 pause

Выбор сервера dwarfpool

Для выбора наиболее подходящего для вас сервера в зависимости от вашего местоположения используйте:


Европейский сервер: eth-eu.dwarfpool.com (Франция) 
Американский сервер: eth-us.dwarfpool.com (Монреаль, Канада) 
Американский сервер: eth-us2.dwarfpool.com (Лас-Вегас) 
RU-Сервер: eth-ru.dwarfpool.com (Москва, Россия) 
ASIA-сервер: eth-asia.dwarfpool.com (Сингапур)
Порт: 80, 8000 и Stratum-Port (прокси + клеймор): 8008
CN-сервер: eth-cn.dwarfpool.com (Шанхай / 上海市) 
CN-сервер: eth-cn2.dwarfpool.com (Пекин / 北京)
Только Stratum-Port (прокси + клеймор): 8008
SG-сервер: eth-sg.dwarfpool.com (Сингапур) 
AU-сервер: eth-au.dwarfpool.com (Мельбурн) 
RU-сервер: eth-ru2.dwarfpool.com (Москва, Россия) 
HK-сервер: eth- hk.dwarfpool.com (Гонконг / 香港)

В этом примере мы использовали российские сервера, так что теперь внутри вашего текстового файла вы должны иметь:

setx GPU_FORCE_64BIT_PTR 0

setx GPU_MAX_HEAP_SIZE 100

setx GPU_USE_SYNC_OBJECTS 1

setx GPU_MAX_ALLOC_PERCENT 100

setx GPU_SINGLE_ALLOC_PERCENT 100

EthDcrMiner64.exe -epool eth-ru2.dwarfpool.com:8008 -ewal Ваш_Ethereum_Address.имя_фермы -epsw x

Замените «Ваш_Ethereum_Address» на свой адрес кошелька Ethereum (который начинается с 0x…), скопируйте его из другого текстового файла и вставьте его здесь.

Замените «имя_фермы» на любое имя, которое вы хотели бы создать для своего майнера. Мы будем использовать «MiningRig1» для нашего примера, так что теперь у вас есть:

setx GPU_FORCE_64BIT_PTR 0

setx GPU_MAX_HEAP_SIZE 100

setx GPU_USE_SYNC_OBJECTS 1

setx GPU_MAX_ALLOC_PERCENT 100

setx GPU_SINGLE_ALLOC_PERCENT 100

EthDcrMiner64.exe -epool eth-ru2.dwarfpool.com:8008 -wal 0x9147460980c93629e775783148591b7d0a0cbf2d.MiningRig1 -epsw x

Сохраните ваш файл как пакетный файл с расширением bat. Для этого выберите «Файл»> «Сохранить как», а затем в поле выберите его имя и введите «.bat» после имени. В поле «Тип файла» выберите «Все файлы» и нажмите «Сохранить».

Поздравляем! Теперь у вас есть готовый пакетный файл для dwarfpool! Создайте ярлык для нового Bat файла и отправьте его на рабочий стол.
