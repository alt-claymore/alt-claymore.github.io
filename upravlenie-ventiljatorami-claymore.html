Claymore - настройка вентиляторов

Для управления вентиляторами в программе используются команды: -fanmin, -fanmax и -tt

Параметр -fanmin

Параметр -fanmin - позволяет установить минимальную скорость вращения вентилятора в процентах.

Например, -fanmin 40 - будут установлены минимальные скорости вентиляторов до 40%. Вы также можете указать значения для каждой карты, например - fanmin 40,55,65.

Эта опция работает только в том случае, если майнер управляет охлаждением, т. е. когда опция -tt используется для задания целевой температуры.

Примечание: для карт NVIDIA эта опция поддерживается только в Windows.

Значения fanmin

  1. Значение по умолчанию - -fanmin 0.
  2. -fanmin число - указать любое число от 0 до 100

Параметр -fanmax

Параметр -fanmax - позволяет установить максимальную скорость вращения вентилятора в процентах.

Например, -fanmax 70 - будут установлены максимальные скорости вентиляторов до 70%. Вы также можете указать значения для каждой карты, например - fanmax 70,85,65.

Значения fanmax

  1. Значение по умолчанию - -fanmax 100.
  2. -fanmax число - указать любое число от 0 до 100

Эта опция также работает только в том случае, если майнер управляет охлаждением, т. е. когда опция -tt используется для задания целевой температуры.

Примечание: для карт NVIDIA эта опция поддерживается только в Windows.

Параметр -tt

Параметр -tt - позволяет установить установить целевую температуру GPU. Например, -tt 83 означает температуру 83C. Вы также можете указать значения для каждой карты, например -tt 83,74,78.

Значения -tt

-tt 1 (по умолчанию) не управляет вентиляторами, но показывает температуру GPU и состояние вентилятора каждые 30 секунд. Укажите значения 2..5 если это слишком часто.

Примечание: для карт NVIDIA в Linux OS управление температурой не поддерживается, поддерживается только мониторинг температуры.

Примечание: для драйверов Linux gpu-pro майнер должен иметь корневой доступ для управления вентиляторами, иначе будет доступен только мониторинг.

Вы можете установить статическую скорость вентилятора, если вы укажете отрицательные значения, например -tt -60 устанавливает 60% скорости вентилятора.

Укажите ноль, чтобы отключить управление и скрыть статистику GPU.

Примеры использования

Пример 1. Стремиться поддерживать температуру 79°:

EthDcrMiner64.exe -epool eth-eu2.nanopool.org:9999 -ewal 0x1a0e2c4cd699cee12672adc223fdb30b93253eba -eworker claymore -epsw x -mode 1 -r 0 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -tt 79 -fanmin 30 -fanmax 80
pause

Программа будет Стремиться поддерживать температуру 79° (параметр -tt 79), минимальная скорость вращения вентиляторов составляет 30% (параметр -fanmin 30), а максимальная скорость - 80% (параметр -fanmax 80)

Пример 2. стремиться поддерживать температуру 79° и остановить при достижении макс. температуры:

EthDcrMiner64.exe -epool eu1.ethermine.org:4444 -ewal 0x9147460980c93629e775783148591b7d0a0cbf2d -eworker asus -epsw x -mode 1 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -tt 79 -fanmin 40 -fanmax 85 -tstop 89
pause

Программа будет стремиться поддерживать температуру 79°, минимальная скорость вращения вентиляторов составляет 40%, а максимальная - 85% и остановится при достижении температуры в 89° (-tstop 89)

Пример 3. Задать постоянную (статическую) скорость вращения вентиляторов

EthDcrMiner64.exe -epool europe.ethash-hub.miningpoolhub.com:20535 -ewal rgzrgz_leo.rgzrgz_leo -epsw x -mode 1 -dbg -1 -mport 0 -esm 2 -etha 0 -retrydelay 1 -tt -80
pause

Программа будет всегда держать обороты вентиляторов в 80%. (-tt -80)

Пример 4. Стремиться поддерживать температуру в 79° и снизить интенсивность майнинга при достижении 85°

EthDcrMiner64.exe -epool eth-ru2.dwarfpool.com:8008 -ewal 0x9147460980c93629e775783148591b7d0a0cbf2d.claymore -mode 1 -r 0 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -tt 79 -ttli 85 -tstop 89 -fanmin 45 -fanmax 83
pause

Программа будет стремиться поддерживать температуру 79°, минимальная скорость вращения вентиляторов составляет 45%, а максимальная - 83% и остановится при достижении температуры в 89° (-ttli 85)

Пример 5. Стремиться поддерживать температуру в 79°, снизить интенсивность майнинга при достижении 85°, остановить при 89° и запустить снова когда карты остынут до 75°

EthDcrMiner64.exe -epool eth.f2pool.com:6688 -ewal rgz -eworker rgz -epsw x -mode 1 -r 0 -dbg -1 -mport 0 -etha 0 -retrydelay 1 -ftime 55 -asm 2 -allpools 1 -tt 79 -ttli 85 -tstop 89 -tstart 75 -fanmin 35 -fanmax 86
pause

Программа будет стремиться поддерживать температуру 79°, минимальная скорость вращения вентиляторов составляет 35%, а максимальная - 86%,остановится при достижении температуры в 89° и запуститься снова при 75° (-tstart 75)
